    Running the App

    Follow the instructions below to run the app in your local environment.

    Requirements

    • Node.js
    • PubNub Account (Free)

    PubNub Account

    To run this application you must obtain publish and subscribe keys from your PubNub Account. If you don't already have an account, you can create one for free.

    1. Sign in to your PubNub Dashboard.

    2. Click Create New App.

    3. Give your app a name, and select Chat App as the app type.

    4. Click Create.

    5. Click your new app to open its settings, then click its keyset.

    6. Enable the Objects feature. Presence and PubNub Functions should have been enabled when the keyset was created.

    7. Select a region to store your user data (e.g. US East).

    8. Locate the Publish and Subscribe keys. You'll need these keys to include in this project.

    Running the project

    1. Clone the GitHub repository.

      git clone https://github.com/pubnub/typescript-ref-app-team-chat-private.git

    2. Install the project.

      cd typescript-ref-app-team-chat
npm install

    3. Run the project in your local environment. If you are running the app for the first time, enter your PubNub keys to begin populating sample data.

      npm start

      A web browser should automatically open http://localhost:3000, and you can explore your very own Team Chat app!
