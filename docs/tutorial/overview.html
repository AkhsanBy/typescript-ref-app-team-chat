    Overview

    This tutorial convers how Team Chat’s key features are put together, and how you can add them in your own applications. Each section focuses on one feature of the application. Although the sections build on each other, feel free to skip around and explore the features that interest you.

    • Theming to quickly customize the look and branding of your application
    • Messaging powered by PubNub
    • Join Conversations
    • Leave Conversations
    • Conversation Members including online/offline status using Presence
    • Typing Indicators let users know when another user is typing
    • Profanity Filtering to moderate messages with serverless PubNub functions
